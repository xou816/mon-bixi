# mon-bixi web extension

![demo.gif](demo.gif)

## Important disclaimer

This extension is in no way associated with Bixi.com. This represents my own vision of a Spotify Wrapped-like feature for Bixi.

## Building

Tested with npm 10.x/11.x, node 22.x/25.x, generic Linux.

A devcontainer configuration is present to help with development.

```bash
npm i # install dependencies
npm run dev # to open the extension using web-ext, unsupported in devcontainer
npm run build # to build the extension in dist/
```

## Usage

Install the extension in your browser.

Log in to secure.bixi.com and profit.
